## Examples

```go
type Person struct {
    Name    string
    Age     int
    Address struct {
        City  string
        State string
    }
}

p := Person{Name: "John Doe", Age:  30}
p.Address.City = "Sprintfield"
p.Address.State = "IL"

md := mdencode.Marshal(p)
fmt.Println(md)

// Output:
// # Person
//
// - **Name**: John Doe
// - **Age**: 30
//
// ## Address
//
// - **City**: Springfield
// - **State**: IL
```

---

```go
var p Person
err := mdencode.Unmarshal(md, &p)
```
